# 🧰 Windows Clean Consumer Setup

One-click setup voor een **verse Windows 11 installatie**.  
Installeert automatisch de belangrijkste apps voor dagelijks gebruik — zonder development tools of overbodige rommel.

---

## 🚀 Quick start

```bash
git clone https://github.com/mwar/scriptjes.git
cd scriptjes/windows-clean-consumer-setup
install.bat
```

## Installeert
- Google Chrome
- 1Password
- Microsoft 365
- Netflix
- Disney+
- WhatsApp
- Spotify
- VLC Media Player
- 7-Zip
- Adobe Acrobat Reader

## Gebruik
1. Download of clone deze repo
2. Ga naar scriptjes\windows-clean-consumer-setup
2. Rechtsklik op `install.bat`
3. Kies uitvoeren

Of dubbelklik simpelweg op `install.bat`.

## Opmerkingen
- Administratorrechten zijn nodig
- Op een verse Windows 11-installatie kan WinGet soms pas na de eerste login goed beschikbaar zijn
- Sommige Microsoft Store-apps kunnen afhankelijk van je Windows/Store-configuratie nog een Store- of accountprompt geven
